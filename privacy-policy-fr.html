2. QUELLES DONNÉES JE COLLECTE ET POURQUOI

Le traitement est différent pour les utilisateurs gratuits et Premium :

A) POUR TOUS LES UTILISATEURS (gratuits et Premium) :

  • Données techniques automatiques : Type d'appareil, système d'exploitation, version de l'Application, adresse IP (anonymisée), heures d'utilisation, fonctions utilisées. Ces données NE vous identifient pas personnellement.
  • Données publicitaires (version gratuite uniquement) : L'Application affiche des annonces via Google AdMob et Meta Audience Network. Pour afficher des annonces pertinentes, l'ID publicitaire de l'appareil (AAID/IDFA) et des données générales d'utilisation peuvent être collectés. Vous pouvez désactiver la personnalisation des annonces dans les paramètres de votre appareil.
  • Autorisations de l'appareil :
    • Stockage/Photos (optionnel) : Requis pour enregistrer les PDF des devis sur votre appareil et les partager (WhatsApp, email, etc.) et pour ajouter un logo depuis la galerie ou l'appareil photo. Les photos utilisées comme logo ne sont JAMAIS téléchargées sur nos serveurs ni sauvegardées dans le cloud ; elles sont utilisées uniquement localement pour générer le PDF.

B) UNIQUEMENT POUR LES UTILISATEURS PREMIUM :

  • Compte : Pour sauvegarder les devis dans le cloud et les synchroniser entre appareils, vous devez créer un compte avec une adresse email. Nous ne stockons que l'email pour l'accès.
  • Données des devis (fournies volontairement par VOUS) : Lorsque vous créez un devis et choisissez de le sauvegarder dans le cloud, les données que vous saisissez sont stockées sur les serveurs sécurisés de Google Firebase (Firestore). Cela peut inclure :
    • Vos informations : Nom, prénom, numéro de téléphone, Numéro de TVA, Code fiscal (Italie), CPF/CNPJ (Brésil), RUT (Chili), CUIT/CUIL (Argentine), email professionnelle, logo.
    • Informations du client : Nom, prénom, téléphone, TVA/Code fiscal/CPF/CNPJ/RUT/CUIT, adresse.
    • Détails du devis : Services, prix, TVA/VAT/IVA/IGV, notes.
Important : En tant qu'utilisateur GRATUIT, aucune donnée personnelle ne quitte votre appareil. Tout est sauvegardé localement (stockage local) et vous êtes le seul à y avoir accès.

4. AVEC QUI PARTAGEONS-NOUS LES DONNÉES

Nous ne vendons JAMAIS vos données. Nous les partageons uniquement avec des fournisseurs essentiels ("Sous-traitants") :

  • Google Firebase (Google Cloud Platform) : Héberge la base de données (Firestore) où les utilisateurs Premium sauvegardent leurs devis. Les serveurs sont sécurisés et situés dans l'UE/États-Unis selon la région. Politique de confidentialité de Google.
  • Google AdMob & Meta Audience Network : Pour gérer les annonces publicitaires. Politique de confidentialité de Meta.
  • Boutiques de paiement : Si vous achetez le Premium via l'Apple App Store ou Google Play Store, ils gèrent les données de paiement. Je ne reçois jamais votre carte de crédit.
  • RevenueCat (optionnel) : Je pourrais utiliser ce service pour gérer les abonnements Premium. Politique de confidentialité de RevenueCat.

Avec tous ces fournisseurs, nous avons des accords contractuels qui les obligent à protéger vos données et à les utiliser uniquement selon nos instructions.

5. INTELLIGENCE ARTIFICIELLE (IA) - GOOGLE GEMINI

L'Application offre une fonctionnalité optionnelle d'analyse IA des devis et listes de prix via Google Gemini API (service d'intelligence artificielle de Google).

Ce qui est envoyé à l'IA :

  • Prix et descriptions des travaux saisis dans le devis/la liste de prix
  • Secteur marchand détecté (ex. construction, plomberie, électricité)
  • Pays et devise sélectionnés par l'utilisateur
  • Nombres et quantités des travaux (uniquement à des fins d'analyse)

Ce qui n'est JAMAIS envoyé à l'IA :

  • Données personnelles (nom, prénom, email, téléphone, numéro de TVA, adresse)
  • Données du client (nom, téléphone, adresse)
  • Coordonnées géographiques ou position précise
  • Données de paiement ou informations financières sensibles
  • ID d'appareil ou identifiants uniques

Comment ça fonctionne :

  • L'analyse IA est uniquement pour les utilisateurs Premium (après achat de l'abonnement)
  • L'utilisateur doit cliquer activement sur le bouton "Analyser avec IA"
  • Aucune analyse automatique n'est effectuée sans votre consentement explicite
  • Les analyses sont traitées en temps réel et ne sont pas stockées par Google Gemini après traitement
  • Seuls les journaux anonymes (date, nombre de requêtes, succès/erreur) sont sauvegardés pour l'amélioration du service

Où les données sont-elles traitées :

Les requêtes à l'IA sont gérées via Google Cloud Functions et Google Gemini API, qui peuvent traiter les données sur des serveurs situés :

  • États-Unis (région us-central1, Iowa)
  • Europe (région europe-west3, Francfort) - préféré pour les utilisateurs UE/EEE/UK/Suisse

Les transferts de données vers les États-Unis sont effectués conformément aux Clauses Contractuelles Types (CCT) de la Commission Européenne et au Data Privacy Framework UE-États-Unis.

Limitations :

  • Les analyses IA sont limitées à 5 pendant les 7 premiers jours d'abonnement Premium, puis 50 par mois
  • La limite se réinitialise automatiquement chaque mois
  • Vous pouvez voir combien d'analyses il vous reste directement dans l'Application

Désactivation :

La fonctionnalité IA est complètement optionnelle. Si vous ne souhaitez pas l'utiliser, il suffit de ne pas cliquer sur le bouton. L'Application fonctionne normalement sans l'analyse IA.

Plus d'informations :

Pour plus de détails sur la façon dont Google gère les données lorsque vous utilisez Gemini, consultez :

  • Politique de confidentialité de Google
  • Conditions de traitement des données Google Cloud

Nos fournisseurs peuvent traiter des données dans des pays en dehors de votre région :

  • Google (États-Unis) : Les transferts sont basés sur les Clauses Contractuelles Types (CCT) de la Commission Européenne (Décision 2021/914) et le Data Privacy Framework UE-États-Unis (adéquation 10.07.2023, Commission Européenne).
  • Meta (États-Unis) : Également basé sur les CCT et le Data Privacy Framework.
  • LGPD (Brésil) : Les transferts sont effectués vers des pays ayant un niveau de protection adéquat reconnu par l'ANPD ou avec des garanties contractuelles spécifiques conformément aux articles 33 et 34 de la LGPD.
  • POPIA (Afrique du Sud) : Les transferts respectent les conditions de la section 72 de la loi POPIA et les autorisations du régulateur de l'information.
  • Argentine (Ley 25.326) : Les transferts respectent le décret 1558/2001 et les autorisations de la Dirección Nacional de Protección de Datos Personales.
  • Chili (Ley 19.628) : Les transferts respectent les normes de protection adéquates prévues par la législation chilienne.

7. CONSERVATION DES DONNÉES (COMBIEN DE TEMPS NOUS LES CONSERVONS)

  • Utilisateurs gratuits : Les données sauvegardées localement restent sur votre appareil jusqu'à ce que vous supprimiez l'Application ou les données. Nous ne les conservons pas.
  • Utilisateurs Premium : Vos données (email et devis) sont conservées tant que vous maintenez votre compte actif. Lors de la suppression du compte, les données sont supprimées de nos serveurs dans :
    • 30 jours pour le RGPD et les normes européennes
    • 15 jours ouvrables pour la LGPD (Brésil) sur demande
    • Délai raisonnable pour la POPIA (Afrique du Sud) et les lois sud-américaines
  • Données techniques anonymes : Conservées pour un maximum de 24 mois pour des analyses agrégées.
  • Données pour obligations légales : Conservées pour la période requise par les lois fiscales et commerciales de chaque pays (typiquement 5-10 ans).

9. SÉCURITÉ DES DONNÉES

Nous mettons en œuvre des mesures techniques et organisationnelles pour protéger vos données :

  • Règles de sécurité Firebase : Accès aux données uniquement pour les utilisateurs autorisés
  • Chiffrement : Données en transit (HTTPS/TLS) et au repos (chiffrement Firestore)
  • Accès limité : Moi seul (Alessio Di Pede) ai un accès administratif
  • Évaluation périodique des risques et mesures appropriées
  • Sauvegardes chiffrées sur des serveurs sécurisés

Cependant, aucun système n'est immunisé à 100%. En cas de violation de données présentant un risque pour vos droits :

  • RGPD : Notification à l'autorité dans les 72 heures et à vous sans délai injustifié
  • LGPD : Communication à l'ANPD et aux personnes concernées dans un "délai raisonnable"
  • POPIA : Notification au régulateur de l'information et aux personnes concernées immédiatement
  • Argentine : Notification à la Dirección Nacional de Protección de Datos Personales

10. COOKIES ET TECHNOLOGIES SIMILAIRES

L'Application n'utilise pas de cookies traditionnels. Nous utilisons :

  • Stockage local : Pour sauvegarder les préférences et les devis localement (sur votre appareil)
  • Identifiants d'appareil : Pour les analyses anonymes (Crashlytics) et la publicité (AdMob)
  • Pixels de suivi : Uniquement pour mesurer les performances des annonces (Meta, Google)

Vous pouvez gérer les préférences de personnalisation des annonces dans les paramètres de votre appareil :

  • iOS : Réglages > Confidentialité > Suivi
  • Android : Réglages > Google > Annonces > Réinitialiser l'ID publicitaire / Désactiver la personnalisation des annonces
  • Appareils Huawei : Réglages > Confidentialité > Publicité et confidentialité

11. MINEURS

L'Application n'est pas destinée aux mineurs selon les âges minimaux suivants :

  • UE/EEE/UK : 16 ans (RGPD) ou âge inférieur si prévu par l'État membre
  • Brésil : 18 ans (LGPD) ou 16-17 ans avec consentement parental
  • Afrique du Sud : 18 ans (POPIA)
  • Argentine : 18 ans (Ley 25.326)
  • Chili : 18 ans (Ley 19.628)

Nous ne collectons pas sciemment les données de mineurs. Si vous êtes parent et découvrez que votre enfant nous a fourni des données, contactez-moi pour une suppression immédiate.
